# amin-GPT 🌍

Welcome to **amin-GPT** — a sleek, intuitive, and highly responsive conversational agent powered by the brilliant **Google Gemma** foundational model. 

## Vision
The goal of this project is to create an immersive, distraction-free environment for interacting with one of the most capable open-weights AI models available today. Whether you are brainstorming ideas, needing quick translations, or requesting creative generation, amin-GPT handles it seamlessly through the sheer cognitive power of the Gemma architecture.

## Core Highlights
- **Powered by Gemma**: Harnesses state-of-the-art conversational intelligence and reasoning.
- **Natural Multilingual Support**: Flawlessly understands and replies natively in Arabic, English, and French.
- **Expressive Persona**: Designed to feel warm, human, and fundamentally conversational.
- **Fluid Experience**: Features a gorgeous, edge-to-edge dark workspace heavily optimized for focus.

## Running Locally
1. Clone the repository to your machine.
2. Ensure your local server environment (like XAMPP) is actively running to handle local traffic routing.
3. Open a terminal in the folder and execute `npm install`.
4. Run `npm run dev` to launch the immersive interface.
